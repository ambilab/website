# Ambilab Website

Bilingual marketing website and news for Ambilab.

## Run

```bash
pnpm install
pnpm dev
```

## Testing

- **Unit tests** (Vitest): `pnpm test` or `pnpm test:run`
- **E2E tests** (Playwright): `pnpm build && pnpm test:e2e`
- **Translation validation**: `pnpm validate:translations`
